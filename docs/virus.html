EchoBreak

Вирус

Инструкция

1. По кнопке снизу скачиваем файл

2. Заходим в загрузки

3. Нажимаем правой кнопкой мыши, Open in terminal

4. Пишем ./EchoBreak_user scan (для сканирования) (чтобы подключиться открываем ещё один терминал Ctr+Alt+T и пишем туда ssh student@ip ip - это тот ip адрес который нашел EchoBreak, пример: ssh student@172.17.212.1) все команды будут выведены если просто в терминале написать ./EchoBreak

Скачать ZIP

Возможности вируса:

Уже сейчас

  • scan - поиск всех доступных компьютеров
  • shutdown -a - выключение всех компьютеров по времени
  • connect - подключение к хосту

В разработке

  • connect [name] - подключение к отдельному компьютеру по имени
  • class - выбор всех компьютеров в 1 из школьных классов
  • login - вход в систему вируса как пользователь
  • stats - статистика всех компьютеров и выполняемых процессов
